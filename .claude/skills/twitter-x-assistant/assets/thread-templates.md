# Thread Templates

## Classic Thread Structure (7-10 tweets)

```
Tweet 1: Big bold hook
Tweet 2: Set the stage (context)
Tweet 3: Raise the stakes (why it matters)
Tweets 4-9: Unfold the drama/insights
Tweet 10: Punchy lesson
Tweet 11: Soft CTA
Tweet 12: Final CTA or engagement ask
```

## Thread Best Practices

### Structure
- **Optimal length**: 7-10 tweets
- **Hook**: Make it bold and attention-grabbing
- **Body**: One idea per tweet, clear and concise
- **Visuals**: Insert images/charts every 3-4 tweets (45% higher completion)
- **CTA**: End with a question or call-to-action

### Writing Tips
- **Use line breaks**: Makes tweets easier to scan
- **Number your tweets**: Helps readers follow along (1/, 2/, etc.)
- **One idea per tweet**: Don't cram multiple points
- **Vary sentence length**: Creates rhythm and flow
- **End with engagement**: Question, poll, or request

### Visual Strategy
- Add relevant images every 3-4 tweets
- Use screenshots of key data/results
- Include charts or infographics
- Keep visuals on-brand
- Increases completion rate by 45%

### Timing
- **Best days**: Tuesday-Thursday
- **Best times**: 9-11 AM EST, 1-3 PM EST
- **Engage heavily**: Reply to comments in first hour
- **First 60 minutes**: Critical for algorithmic boost

### Engagement Tactics
- Ask questions throughout thread
- Request saves: "Bookmark for later"
- Encourage replies: "Which point resonates most?"
- Tag relevant people (sparingly)
- Reply to every comment in first hour
